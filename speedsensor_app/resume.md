- Обязателен `#include <Adafruit_TinyUSB.h>`
- Датчик логически инвертирован: HIGH = луч свободен, LOW = луч перекрыт

## Прошивка (sensor_counter.ino)

Вывод в Serial каждые 500 мс:
```
Pulses: N | RPS: X.XXX | RPM: X.X | V: X.X mm/s | Pin: HIGH/LOW
```
Команды: `t`, `r`, `i`, `h`

Формула: `rps = pulseCount / SLOTS / (elapsed_ms / 1000.0)`

## core/motor_sim.py — актуальное состояние

```python
class MotorSimModel:
    # Параметры двигателя
    max_rpm: float = 600.0        # макс. обороты (RPM) при реостате 100%
    # max_rps — производное: max_rpm / 60

    # Параметры диска датчика
    disk_diameter_mm: float = 75.0
    slots: int = 20
    slot_width_mm: float = 5.0    # ширина пропила
    slot_gap_mm: float = 6.0      # ширина перемычки

    # Реостат
    rheostat_pct: float = 0.0     # 0–100%

    # Шум
    noise_percent: float = 1.0
```

Производные свойства: `max_rps`, `target_rps`, `target_omega`,
`slot_period_mm`, `disk_circumference_mm`

## ui/sim_settings_panel.py — актуальное состояние

Панель симуляции содержит три группы:
1. **Положение реостата** — вертикальный слайдер 0–100%, кнопка «Стоп»
2. **Параметры двигателя** — поле `max_rpm` (RPM), отображение `max_rps`
3. **Диск датчика угловой скорости** — диаметр, кол-во пропилов, ширина пропила,
   ширина перемычки; вычисляемые: период решётки, длина окружности;
   предупреждение если пропилы не вмещаются на диск

Вся панель обёрнута в `QScrollArea`.

## ui/experiment_widget.py — актуальное состояние

### Toolbar (слева → справа)
`Старт` | `Стоп` | `Очистить` | stretch | `Единицы` | `Окно (с)` |
`sep` | `Масштаб X` | `Масштаб Y` | `sep` | `🖊 Вид линии` | `sep` | `📍 Tooltip`

### Строка значений (под toolbar)
- Крупное текущее значение ω (всегда видно)
- Статичный блок мгновенных значений (виден только когда Tooltip выкл.)

### График (pyqtgraph)
- Скользящее окно по X, автомасштаб Y
- Пунктирная линия уставки (только в режиме симуляции)
- Кеш numpy-массивов (`_t_cache`, `_y_cache`, `_y_raw_cache`), флаг `_cache_dirty`
- Прореживание до 800 точек в видимом окне

### Tooltip (перекрестие + плавающее окошко)
- Режим **follow** (`_tooltip_follow = True`, чекбокс «📍 Tooltip» включён):
  - `_GraphTooltip` — дочерний виджет `PlotWidget` (без `Tool`-флага!)
  - Позиция через `pw.mapFromScene(pos)`, клип по всем 4 краям
  - Показывает: `t`, `ω`, `об/с`, `RPM`, `V мм/с`
- Режим **static** (чекбокс выключен):
  - Плавающий tooltip скрыт
  - `_static_info` (QLabel в строке значений) обновляется в `_on_sample`

### Настройка вида линии (`_LineStyleDialog`)
Параметры в `_line_params`: `color`, `width`, `style` (Qt.PenStyle),
`symbol` (scatter), `symbol_size`

### Смена единиц
После смены — `_cache_dirty = True` + `_refresh_plot()` + `vb.enableAutoRange()`

### Thread-safety
`new_sample` подключён через `Qt.ConnectionType.QueuedConnection`
