## Прошивка (sensor_counter.ino)

### Вывод в Serial каждые 100 мс (REPORT_MS=100):
```
Pulses: N | RPS: X.XXX | RPM: X.X | V: X.X mm/s | Pin: HIGH/LOW
```

### Команды Serial:
| Команда | Доступ | Действие |
|---|---|---|
| `t` | открытый | тест пина |
| `r` | открытый | сброс счётчика |
| `i` | открытый | расширенный вывод: имя, назначение, сценарии + параметры диска |
| `h` | открытый | помощь |
| `W:<pw>:<key>:<value>` | только с паролем | записать поле идентификации во flash |

Формула: `rps = pulseCount / SLOTS / (elapsed_ms / 1000.0)`

### Идентификация датчика (хранится в InternalFS / flash)

| Поле | Ключ в файле | Пример значения |
|---|---|---|
| Название датчика | `SENSOR_NAME` | `Датчик угловой скорости` |
| Назначение | `SENSOR_PURPOSE` | `Измерение угловой скорости` |
| Сценарии | `SENSOR_SCENARIOS` | `1,2,3` |

**Файл на flash:** `/sensor_meta.txt` (InternalFS, Adafruit nRF52)

**Формат файла:**
```
SENSOR_NAME=Датчик угловой скорости
SENSOR_PURPOSE=Измерение угловой скорости
SENSOR_SCENARIOS=1,2,3
```

**Защита паролем:**
- Пароль задаётся как `#define ADMIN_PASSWORD "..."` на этапе компиляции
- По Serial пароль никогда не передаётся в открытом виде
- Команда записи: `W:<пароль>:<ключ>:<значение>`
- Пример: `W:secret123:SENSOR_NAME:Датчик угловой скорости`
- При неверном пароле: молчание (нет ни ошибки, ни подтверждения)
- ПО для работы с датчиком не имеет доступа к записи — только чтение через команду `i`

**Вывод команды `i` (новый формат):**
```
--- SENSOR INFO ---
Name: Датчик угловой скорости
Purpose: Измерение угловой скорости
Scenarios: 1,2,3
--- DISK INFO ---
Диаметр: 75 мм
Прорезей: 20
Прорезь: 5.00 мм
Промежуток: 6.00 мм
Период: 11.00 мм
-----------------
```

### PortInfo (расширен для идентификации датчика)

```python
@dataclass
class PortInfo:
    device: str
    description: str
    vid: int | None
    pid: int | None
    confirmed: bool = False
    sensor_name: str = ""        # парсится из "Name: ..."
    sensor_purpose: str = ""     # парсится из "Purpose: ..."
    sensor_scenarios: str = ""   # парсится из "Scenarios: ..."
```

- `port_scanner._probe_port` парсит строки `Name:`, `Purpose:`, `Scenarios:` из ответа на `i`
- Данные отображаются в информационном окне при подключении
- Запись данных в датчик из ПО — **невозможна**
- Реализация изменений ПО — **в отдельном чате**

### Маркеры идентификации для probe

```python
PROBE_MARKER = ("speedsensor", "rps:", "pulses:", "name:", "purpose:")
```
